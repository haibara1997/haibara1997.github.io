Leetcode刷题日记85最大矩形

发表于 2020-12-28 分类于 算法 阅读次数：



一种优化过的暴力解法：
创建一个矩阵，该矩阵记录着左边数起连续有多少个1
如针对题目中的矩阵，创建矩阵如下

- - - - -
1 0 1 0 0
1 0 1 2 3
1 2 3 4 5
1 0 0 1 0

然后对于每个点，计算以这个点为右下角最大可以创建多大的矩阵
如对于matrix[2][4]，行数是2(宽为2)，列数是$ min(5,3)=3 $，即长为3，所以矩阵的大小为$ 2\times 3=6 $
